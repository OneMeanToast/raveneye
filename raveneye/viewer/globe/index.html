The viewer is a static page that loads scenario.json via fetch(). Browsers block this on file://; serve the directory:

python3 -m http.server

then open http://localhost:8000/.

Override URL with ?scenario=path/to/scenario.json or set window.RAVENEYE_SCENARIO_URL.
