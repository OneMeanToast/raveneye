Could not load sim.json

The viewer is a static HTML file but loads sim.json via fetch(), which most browsers block when opened directly from disk. Serve the directory:

python3 -m http.server

then open http://localhost:8000/.

You can also override the URL with ?sim=path/to/sim.json or window.RAVENEYE_SIM_URL.
